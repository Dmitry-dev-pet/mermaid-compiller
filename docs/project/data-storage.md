# Хранение данных

## localStorage

Используются ключи:

- `dc_ai_config` — настройки LLM (провайдер, ключи, endpoint, модель, фильтры).
- `dc_app_state` — состояние UI (тип диаграммы, ширины колонок, тема, язык, fullscreen).
- `dc_active_session_id` — активная сессия истории.

## IndexedDB

База `dc_history` хранит историю шагов и ревизий. Схема описана в `docs/project/history.md`.

## Что НЕ хранится

- Секреты LLM не отправляются на сервер приложения и остаются в браузере.
- История чата/шагов хранится локально в IndexedDB.
