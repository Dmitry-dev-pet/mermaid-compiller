# Troubleshooting

## LLM не подключается

- Проверьте ключ и endpoint в `Header`.
- Для OpenRouter нужен корректный `openRouterKey`.
- Для Cliproxy убедитесь, что сервер доступен и поддерживает `/v1/chat/completions`.

## Модели не отображаются

- В OpenRouter без ключа список моделей не загрузится.
- Проверьте фильтры (free-only, context ≥ 8k).

## Build возвращает пустой результат

- LLM мог вернуть ответ без Mermaid-кода.
- Проверьте ограничения/модель или попробуйте другой prompt.

## Превью не рендерится

- Проверьте, что код валиден (статус в редакторе).
- При ошибке рендера отображается overlay с текстом ошибки.

## Zoom/Pan не работает

- Убедитесь, что SVG успешно отрендерен.
- При смене fullscreen выполняется fit — это может сбрасывать масштаб.

## История не сохраняется

- Проверьте доступность IndexedDB в браузере.
- Очистка localStorage может сбросить активную сессию.
