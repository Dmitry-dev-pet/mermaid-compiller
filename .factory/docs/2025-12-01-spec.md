1. Определить официальный источник документации Mermaid на GitHub (репозиторий `mermaid-js/mermaid`).
2. Считать последнюю опубликованную версию (например, через GitHub Releases API или страницу `package.json` ветки main) и сохранить номер версии.
3. Проверить доступность онлайн-документации, запросив один из ключевых файлов (например, `docs/intro/index.md` из ветки main) и убедившись, что HTTP‑ответ успешный.
4. Вывести найденный номер версии и зафиксировать, доступна ли документация.
5. Если онлайн-доступ недоступен, предусмотреть сообщение об ошибке и инструкцию переключиться на локальную копию.
