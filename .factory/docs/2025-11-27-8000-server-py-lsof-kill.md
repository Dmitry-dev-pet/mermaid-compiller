Чтобы освободить порт 8000 под наш `server.py`, нужно найти процесс, который его держит, и завершить его.

**План действий**
1. Выполнить `lsof -i :8000` в корне проекта (или просто в терминале), чтобы увидеть PID процесса, который слушает порт 8000.
2. Убедиться по имени команды/пути, что это действительно наш старый `server.py` или другой ненужный процесс, а не что‑то системно важное.
3. Завершить этот процесс через `kill <PID>` (при необходимости `kill -9 <PID>`, если обычный kill не сработает).
4. После освобождения порта повторно запустить сервер с нужными переменными окружения, например: `PORT=8000 CONTEXT7_LIBRARY_ID=/mermaid-js/mermaid .venv/bin/python server.py` и снова открыть UI на `http://localhost:8000`.
